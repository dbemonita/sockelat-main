## Sockelat
**Monita 5 HMI / Admin API**

### Install

    > git clone https://guncloud@bitbucket.org/dbe-design/sockelat-main.git 
    > cd sockelat-main
    > nmp install --registry http://project2.daunbiru.com:4873
    
### Config
.env
    
    MYSQL_HOST=localhost
    MYSQL_USER=root
    MYSQL_PASSWORD=root
    MYSQL_PORT=3306
    MYSQL_DATABASE=monita_v5_1
    REDIS_HOST=localhost
    REDIS_PORT=6379
    PORT=5000
    SECRET_KEY=waduhek
    LOKET_INSTANCE=xxx
